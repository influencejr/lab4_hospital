## 25.11
### Змінено структуру призначень:

### Призначення "Видача ліків" (DispensingOfMedicines)
Перероблено за патерном Flyweight:
- створено окремий клас MedicineType, щоб не створювати багато разів призначення для одних і тих самих ліків;
- створено MedicineFactory для перевірки призначення на існування. 

### Створення призначень:
Створено AppointmentFactory для зручного створення нових призначень;
Перероблено конструктори всіх призначень:
- додано ім'я та пошту пацієнта;
- в конструкторі призначення "Видача ліків" додано об'єкт MedicineType;
